🎰 Lottery Generator – Генератор лотерейных билетов 
Мощный инструмент для генерации случайных чисел для различных лотерей (6/49, 5/36, 7/39 и др.).
Поддерживает проверку билетов, статистику выпадений, экспорт в CSV и настраиваемые форматы.
Реализован на 7 языках программирования с единым интерфейсом командной строки.

🚀 Возможности
Гибкие форматы – задавайте количество номеров и диапазон (например, 6 49, 5 36).

Генерация билетов – создание одного или нескольких билетов с уникальными числами.

Проверка выигрыша – сравнение билета с выигрышными номерами.

Статистика – анализ частоты выпадения чисел на основе истории (опционально).

Экспорт – вывод результатов в файл (CSV или текстовый).

Цветной вывод – выделение совпадений и сообщений.

Сохранение истории – запись всех сгенерированных билетов в файл (опционально).

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> <количество_номеров> <максимальное_число> [опции]
Опции
Опция	Описание
-c, --count <N>	Количество билетов для генерации (по умолчанию 1)
-w, --winning <числа>	Выигрышные номера для проверки (через запятую)
-s, --stats	Показать статистику выпадений (если есть история)
-o, --output <file>	Сохранить результат в файл
-v, --verbose	Подробный вывод (показывать каждый билет)
-h, --help	Справка
Примеры
bash
# Генерация одного билета 6/49
python lottery.py 6 49

# Генерация 5 билетов 5/36
python lottery.py 5 36 -c 5

# Проверка билета на выигрыш (номера 1,2,3,4,5,6)
python lottery.py 6 49 -w "1,2,3,4,5,6"

# Сохранить 10 билетов в файл
python lottery.py 6 49 -c 10 -o tickets.txt

🧠 Логика работы
Генерация билета – случайный набор уникальных чисел в заданном диапазоне, отсортированных по возрастанию.

Проверка – подсчёт совпадений с выигрышными номерами.

Статистика – анализ частоты появления чисел в сгенерированных билетах (если ведётся история).

Экспорт – запись результатов в файл в читаемом формате.

✨ Дополнительные фичи
Поддержка пользовательских форматов – любые сочетания чисел и диапазонов.

Цветная подсветка совпадений при проверке.

Автосохранение истории для статистики (в некоторых версиях).
